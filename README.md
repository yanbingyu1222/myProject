# 多功能网页集合

这是一个包含多个实用功能的网页集合，涵盖生日祝福、随机点餐和随机点名等功能：
- 生日快乐.html - 功能丰富的交互式生日祝福页面
- 生日祝福.html - 简洁优雅的生日祝福页面
- 随机点餐.html - 选择困难症的救星，一键随机选择美食
- 随机点名.html - 课堂互动神器，公平公正的随机点名系统

## 本地运行

1. 克隆或下载此项目
2. 直接双击HTML文件在浏览器中打开
3. 或者使用本地服务器：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx http-server -p 8000
   ```

## 使用方法

### 🎂 生日祝福页面
1. 打开生日祝福相关的HTML文件
2. 输入寿星的昵称进行个性化
3. 享受完整的生日祝福体验！

### 🎯 实用工具页面
1. **随机点餐**：
   - 添加或导入菜品列表
   - 点击"随机选择"开始选择
   - 可排除已吃选项，查看历史记录

2. **随机点名**：
   - 导入参与人员名单
   - 点击"开始点名"进行随机选择
   - 查看点名记录确保公平性
