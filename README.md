# 生日快乐网页

这是一个精美的生日祝福网页，包含两个版本：
- 生日快乐.html - 功能丰富的交互式生日祝福页面
- 生日祝福.html - 简洁优雅的生日祝福页面

## 在线访问

- **生日快乐页面**: [https://你的用户名.github.io/brithday/生日快乐.html](https://你的用户名.github.io/brithday/生日快乐.html)
- **生日祝福页面**: [https://你的用户名.github.io/brithday/生日祝福.html](https://你的用户名.github.io/brithday/生日祝福.html)

## 功能特色

### 生日快乐.html
- 🎂 动态生日蛋糕和蜡烛吹灭效果
- 🎨 精美的粒子背景和动画效果
- 🎵 背景音乐控制
- 💝 亲友祝福卡片（3D翻转效果）
- 🌟 许愿墙和心愿记录
- 📱 完全响应式设计
- 🎭 彩带和烟花特效

### 生日祝福.html
- 🎨 简洁优雅的设计
- 📝 个性化昵称输入
- 🎂 生日蛋糕动画
- 🎁 礼物盒开启效果
- 📅 倒计时功能
- 📱 移动端优化

## 本地运行

1. 克隆或下载此项目
2. 直接双击HTML文件在浏览器中打开
3. 或者使用本地服务器：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx http-server -p 8000
   ```
